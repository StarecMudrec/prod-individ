# Твоя игра

Браузерная интеллектуальная игра в духе телевикторин (Vue 3 + TypeScript + Tailwind CSS).

## Требования

- Node.js 20 LTS
- npm

## Запуск локально

```bash
npm ci
npm run build
npm start
```

После этого приложение доступно по адресу из вывода `npm start` (обычно http://localhost:4173).

Режим разработки с горячей перезагрузкой:

```bash
npm run dev
```

## Сборка

- Сборка статики в папку `public`: `npm run build`
- Запуск превью собранного приложения: `npm start` (или `npm run preview`)

## Экстрафичи

- **Импорт и экспорт пакетов вопросов**: на экране «Редактор вопросов и пакетов» (`/packs`) можно экспортировать текущий
  набор вопросов в JSON-файл и импортировать новые пакеты из файла. При импорте структура пакета валидируется, а при
  ошибках показывается список проблем.
- **Редактор тем и вопросов**: встроенный редактор (`/packs/editor`) позволяет изменять темы и вопросы активного пакета,
  добавлять новые темы и вопросы, а также редактировать специальные вопросы («Кот в мешке» и «Аукцион») и финальный вопрос.
